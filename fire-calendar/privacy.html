Коротко

Основные данные (календарь/заметки) — у тебя на устройстве. Геолокация — только по запросу. Разработчик не продаёт персональные данные.

1. Общие положения

Настоящая Политика конфиденциальности описывает, какие данные могут обрабатываться при использовании мобильного приложения «Служебный календарь». Используя приложение, пользователь подтверждает согласие с условиями настоящей Политики.

2. Какие данные могут обрабатываться

2.1. Данные, вводимые пользователем

  • записи календаря;
  • заметки;
  • служебные напоминания.

Эти данные хранятся локально на устройстве пользователя либо в используемых облачных сервисах, если такая функциональность предусмотрена. Регистрация пользователя не требуется.

2.2. Геолокация

Приложение может запрашивать доступ к местоположению только по инициативе пользователя для отображения карты. Геолокация не используется для отслеживания и не передаётся разработчику.

2.3. Технические данные

Автоматически могут обрабатываться обезличенные технические данные:

  • модель устройства;
  • версия iOS;
  • версия приложения;
  • информация о сбоях.

2.4. Уведомления

Для доставки уведомлений используется Apple Push Notification Service. Разработчик не получает доступ к содержанию личных уведомлений пользователя.

3. Реклама

Приложение может показывать рекламу, включая рекламу от третьих лиц (например, Yandex Mobile Ads SDK). Рекламные сервисы могут использовать собственные технологии обработки данных в соответствии со своими политиками конфиденциальности.

4. Использование данных

Данные используются исключительно для:

  • работы приложения;
  • отображения календаря и карты;
  • отправки уведомлений;
  • повышения стабильности;
  • устранения ошибок.

5. Передача данных третьим лицам

Разработчик не продаёт персональные данные пользователей. Передача данных возможна:

  • в рамках работы сторонних сервисов (карты, реклама, уведомления);
  • при наличии законных требований государственных органов.

6. Хранение и защита данных

Применяются стандартные технические меры защиты, предусмотренные используемыми платформами и сервисами.

7. Права пользователя

Пользователь вправе:

  • отключить уведомления в настройках iOS;
  • запретить доступ к геолокации;
  • удалить приложение и связанные с ним данные;
  • связаться с разработчиком по вопросам конфиденциальности.

8. Изменения политики

Политика может обновляться. Актуальная версия всегда публикуется на странице разработчика и в приложении.

9. Контакты
