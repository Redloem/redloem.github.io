Приложение: «Айпи»
Разработчик: Redloem
Дата обновления: 3 июня 2026

Приложение «Айпи» показывает сетевые данные устройства, включая внешний IP-адрес, локальный IP-адрес, тип сети, системный DNS, страну подключения, сведения о провайдере, а также результаты теста скорости интернет-соединения.

1. Какие данные обрабатываются

Приложение может обращаться к внешним сетевым сервисам для определения внешнего IP-адреса, страны, провайдера подключения и выполнения теста скорости. При использовании этих функций вашему устройству необходимо передавать IP-адрес и технические сетевые данные, необходимые для ответа сервиса.

2. Что приложение делает

Приложение предназначено для отображения сетевой информации устройства и диагностики интернет-соединения. Оно может показывать внешний IP-адрес, локальный IP-адрес, DNS-серверы, сведения о провайдере, тип подключения, уровень сигнала и результаты тестов скорости сети.

3. Что приложение не делает

Приложение не предоставляет услуги VPN, не маршрутизирует интернет-трафик пользователя, не создаёт зашифрованные туннели и не изменяет сетевые настройки устройства. Приложение также не продаёт и не передаёт персональные данные пользователей третьим лицам.

4. Сторонние сервисы

Для определения IP-адреса, геоданных, сведений о провайдере и выполнения тестов скорости приложение может использовать сторонние сетевые API. На их стороне могут действовать собственные правила обработки данных.

5. Хранение данных

Настройки интерфейса и другие пользовательские параметры сохраняются локально на устройстве пользователя. Приложение не хранит историю интернет-трафика и не собирает данные о посещаемых сайтах.

6. Ваш контроль

Пользователь может удалить приложение и все связанные с ним локальные данные в любой момент стандартными средствами операционной системы.

7. Связь

Если у вас есть вопросы по обработке данных, используйте контактные данные, указанные в разделе поддержки приложения.
